# Billing Software (Tkinter-based)

A Python-based billing software GUI built with Tkinter, supporting product categories, customer input, and automatic bill generation.

## Features
- Medical, grocery, and cold drink items
- Automatic tax calculation
- Bill save/load functionality
- GUI built with Tkinter

## Requirements
- Python 3.x
- Tkinter (usually included with Python)

## How to Run
```bash
python main.py
